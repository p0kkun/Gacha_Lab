# LINE Developers セットアップ手順

## 1. LINE Developers アカウント作成

1. **LINE Developers コンソールにアクセス**

   - [LINE Developers コンソール](https://developers.line.biz/console/) にアクセス
   - LINE アカウントでログイン

2. **メールアドレスで作成した場合の注意**
   - LINE アプリと連携されていない場合は、LINE アプリから連携する必要があります
   - または、メールアドレスでログインしたアカウントをテストユーザーとして追加

## 2. プロバイダー登録

1. **プロバイダー作成**

   - LINE Developers コンソールの「プロバイダー」ページに移動
   - 「新規作成」をクリック
   - **プロバイダー名**: `Gacha Lab`
   - **説明**: 任意（省略可）
   - 「作成」をクリック

2. **個人の場合**
   - プロバイダー名は個人名でも可
   - 会社情報は不要（個人開発の場合）

## 3. チャネル作成

1. **チャネル作成**

   - プロバイダー選択後、「チャネル」タブを開く
   - 「チャネルを追加」→「LINE ミニアプリ」を選択

2. **基本情報の入力**

   - **チャネル名**: `Gacha Lab テストアプリ`
   - **サービス提供地域**: 日本、タイ、台湾（必要に応じて選択）
   - **メールアドレス**: `your-email@example.com`（実際のメールアドレスに置き換え）
   - **チャネル説明**:

     ```text
     Gacha Lab は、今後有償で実現するためのテストアプリとなります。LINE 上でポーカー大会用の参加チケットを抽選できるテスト用ミニアプリです。ユーザーはガチャボタンを押すだけでランダムに動画演出が再生され、当選時は管理画面から登録したチケット情報が表示されます。
     ```

   - **注意**: 全角括弧（「」）は使用しない（4 バイト文字エラーの原因になる可能性）

3. **規約への同意**

   - LINE 開発者契約
   - LINE ミニアプリプラットフォーム規約
   - LINE ミニアプリポリシー
   - すべてにチェックを入れて同意

4. **チャネル作成完了**
   - 「作成」をクリック
   - チャネル ID をメモ（後で使用）

## 4. LIFF エンドポイント登録

1. **LIFF アプリ作成**

   - チャネル設定画面で「LIFF」タブを開く
   - 「追加」をクリック

2. **LIFF アプリ情報の入力**

   - **LIFF アプリ名**: `Gacha Lab`
   - **サイズ**: `Full`（全画面）
   - **エンドポイント URL（開発用）**:
     - 初期: `https://developers.line.biz/assets/liff-default-dev.html`
     - 後でローカル開発用 URL（ngrok/localtunnel）に変更

3. **LIFF ID をメモ**
   - 作成後に表示される LIFF ID（例: `1234567890-XXXXXXXX`）をコピー
   - 後で`.env.local`に設定します

## 5. テストユーザー追加

1. **開発中チャネルのアクセス**

   - チャネルが「開発中」ステータスの場合、テストユーザーとして追加する必要があります

2. **テストユーザー追加**
   - チャネル設定画面で「ユーザー」タブを開く
   - 「テストユーザーを追加」をクリック
   - 自分の LINE アカウント（ユーザー ID または QR コード）を追加

## 6. サービス利用規約・プライバシーポリシー URL 設定

1. **GitHub Pages で公開**

   - サービス利用規約: `https://YOUR_GITHUB_USERNAME.github.io/Gacha_Lab/terms.html`
   - プライバシーポリシー: `https://YOUR_GITHUB_USERNAME.github.io/Gacha_Lab/privacy.html`

2. **チャネル設定で URL を登録**
   - チャネル設定画面で「基本設定」タブを開く
   - 「サービス利用規約 URL」と「プライバシーポリシー URL」を入力

## 確認項目

- [ ] LINE Developers アカウントを作成した
- [ ] プロバイダー「Gacha Lab」を作成した
- [ ] チャネル「Gacha Lab テストアプリ」を作成した
- [ ] LIFF アプリを作成し、LIFF ID をメモした
- [ ] テストユーザーとして自分の LINE アカウントを追加した
- [ ] サービス利用規約・プライバシーポリシー URL を設定した
