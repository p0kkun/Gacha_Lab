# Gacha Lab 開発手順 - 全体概要

## プロジェクト概要

LINE ミニアプリとして動作するガチャアプリケーション「Gacha Lab」の開発手順です。

### 主な機能

- LINE LIFF SDK を使用したユーザー情報取得
- 全画面オーバーレイでガチャ選択
- 左側メニューでガチャ種類選択（通常ガチャ、プレミアムガチャ）
- 右側に「ガチャを引く」ボタン
- 動画演出による当たり判定表示

### 技術スタック

- **フロントエンド/バックエンド**: Next.js 16 (App Router)
- **UI**: React + Tailwind CSS
- **LINE 連携**: LIFF SDK (@line/liff)
- **開発環境**: Node.js 20+
- **ローカル公開**: ngrok / localtunnel
- **デプロイ予定**: AWS Amplify (GitHub 連携)
